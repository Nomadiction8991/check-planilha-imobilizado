# Passo a Passo: Preview de Importação

## O que é possível executar:
- **Ler o título da revisão**: O topo diz “Escolha as igrejas que devem entrar na importação.”
- **Conferir a administração selecionada**: Abaixo do título aparece uma linha com “Administração: ...”.
- **Ler a tabela principal**: A tabela lista as igrejas detectadas, uma por linha.
- **Ver os totais por igreja**: Em cada linha aparecem os totais de novos, atualizar, exclusões e erros.
- **Abrir a área de dependências**: Quando a linha expande, aparece a subtabela de setores/dependências com os respectivos totais.
- **Ler os controles por igreja**: A linha possui seletor de ação para definir o que entra ou não no lote.
- **Ver o bloco final de confirmação**: No rodapé existe a seção de confirmação final com botão principal para concluir.
- **Voltar antes de confirmar**: Se a composição estiver errada, a pessoa pode não confirmar e ajustar a origem.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o momento de decisão antes da gravação.*
