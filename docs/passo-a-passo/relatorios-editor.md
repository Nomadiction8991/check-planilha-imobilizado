# Passo a Passo: Editor de Células

## O que é possível executar:
- **Editar células diretamente**: A pessoa ajusta dados pontuais sem sair da visualização do relatório.
- **Corrigir valores e textos**: Quando a célula precisa de ajuste fino, a edição acontece ali mesmo.
- **Ver o resultado rapidamente**: A lógica da tela permite revisar e perceber a alteração sem fluxo longo de navegação.
- **Salvar alterações com pouco atrito**: O objetivo é reduzir etapas entre identificar o erro e gravar a correção.
- **Ajustar a forma final do relatório**: O editor existe para dar acabamento ao documento, não para refazer o relatório inteiro.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o ajuste fino da saída final.*
