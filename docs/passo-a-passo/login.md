# Passo a Passo: Tela de Login

## O que é possível executar:
- **Ler o selo de acesso**: No topo do card aparece o selo “Acesso restrito”.
- **Ver o tema**: O cabeçalho tem o controle de tema no canto superior direito.
- **Digitar o e-mail**: O primeiro campo é o input de e-mail, com label acima.
- **Digitar a senha**: O segundo campo é o input de senha, logo abaixo do e-mail.
- **Entrar**: O botão “Entrar” fica no rodapé do formulário e envia as credenciais.
- **Abrir recuperação**: O link “Esqueci minha senha” fica ao lado do botão de entrar.
- **Ler erro ou status**: Se houver erro ou status, a mensagem aparece em um bloco separado acima do formulário.
- **Usar no mobile**: Em telas pequenas o card continua central, mas com largura reduzida e campos empilhados.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o comportamento esperado em uso diário.*
