# Passo a Passo: Recuperar Senha

## O que é possível executar:
- **Solicitar recuperação**: A pessoa informa o email cadastrado para iniciar o processo.
- **Receber instrução de envio**: O sistema envia o link ou a mensagem de recuperação para a caixa de entrada informada.
- **Confirmar o pedido**: A tela mostra que a solicitação foi registrada, para que o usuário saiba que precisa checar o email.
- **Retornar ao login**: Depois da solicitação, o fluxo naturalmente volta para a autenticação, caso o usuário queira tentar entrar de novo.
- **Resolver perda de senha**: Essa tela existe como caminho formal quando o acesso normal não é possível.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o caminho de restauração de acesso.*
