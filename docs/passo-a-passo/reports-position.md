# Passo a Passo: Posição de Estoque

## O que é possível executar:
- **Ler o resumo geral**: A pessoa vê totais, pendências, checados, observações e outros indicadores da igreja selecionada.
- **Explorar métricas combinadas**: Os blocos extras mostram combinações de checagem, observação, etiqueta e edição.
- **Abrir o backup CSV**: O fluxo permite exportar a posição quando a função está disponível.
- **Usar como painel analítico**: Não é uma tela de edição; ela existe para entendimento de estado.
- **Conferir o detalhamento**: A tabela abaixo dos cards serve para ver os itens com mais granularidade.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a leitura analítica da posição.*
