# Passo a Passo: Dashboard

## O que é possível executar:
- **Ler o painel inicial**: O dashboard funciona como porta de entrada e ponto de partida do sistema.
- **Ver o contexto atual**: A tela mostra qual unidade está ativa e qual perfil está logado.
- **Usar atalhos**: Os cards e blocos de navegação levam para as áreas principais do sistema.
- **Ir para outras telas**: O dashboard reduz a necessidade de abrir menus profundos.
- **Retomar o trabalho**: Em uso diário, a tela ajuda a voltar para a operação mais provável com menos clique.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o papel real da tela no uso diário.*
