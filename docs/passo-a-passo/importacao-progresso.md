# Passo a Passo: Progresso de Importação

## O que é possível executar:
- **Ver o título da tela**: O topo informa “Importação em processamento.”
- **Ler o nome do arquivo**: A seção principal mostra o nome do arquivo importado ou a identificação da importação.
- **Ler a administração**: A mesma seção informa qual administração está sendo processada.
- **Acompanhar o status textual**: Existe uma linha “Status” que muda conforme a importação roda.
- **Acompanhar a barra de progresso**: A barra horizontal mostra o avanço visual da operação.
- **Ver a porcentagem**: A linha “Progresso” exibe o percentual completo.
- **Ler os dados ocultos de controle**: A página carrega total de linhas, processadas, sucesso e erro para alimentar o script.
- **Abrir a tela de sucesso**: Quando termina, aparece uma área com a mensagem de conclusão e o total de linhas processadas com sucesso.
- **Ir para produtos**: O botão “Ir para produtos” aparece na área de sucesso.
- **Revisar erros**: Se houver permissão, aparece o botão para revisar erros desta importação.
- **Voltar para a prévia**: Se ocorrer falha, a área de erro mostra um botão para voltar à prévia.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a etapa de monitoramento do processo.*
