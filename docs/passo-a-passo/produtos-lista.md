# Passo a Passo: Listagem de Produtos

## O que é possível executar:
- **Selecionar a igreja**: O primeiro filtro define o recorte do inventário. Isso não é só visual; ele muda a base exibida na listagem.
- **Buscar por texto livre**: O campo “Busca geral” procura por código, descrição, dependência, tipo ou status.
- **Filtrar por dependência**: O seletor de dependência refina a lista para uma dependência específica.
- **Filtrar por tipo de bem**: O seletor de tipo de bem reduz a lista para um tipo específico.
- **Filtrar por status**: O seletor de status mostra todos, novos, editados ou outros estados suportados pelo cadastro.
- **Limpar os filtros**: O botão “Limpar” remove o recorte atual e volta a lista para o estado geral.
- **Abrir a verificação**: Quando permitido, a tela oferece atalho para o checklist de produtos.
- **Criar novo produto**: O botão “Novo produto” leva para o formulário de inclusão.
- **Editar um item**: Cada linha traz o botão “Editar” no campo de ações.
- **Ler as cápsulas de status**: O usuário vê indicadores como 14.1, Nota fiscal, Novo e Editado.
- **Navegar pela paginação**: O rodapé mostra página atual, total de páginas e links anterior/próximo.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o comportamento prático da consulta e manutenção.*
