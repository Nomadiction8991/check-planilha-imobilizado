# Passo a Passo: Logs de Auditoria

## O que é possível executar:
- **Consultar histórico**: A tela permite ver o que usuários fizeram no sistema ao longo do tempo.
- **Filtrar ações**: O usuário pode procurar registros por contexto, usuário, data ou outro recorte disponível.
- **Investigar eventos**: A listagem ajuda a entender quando algo foi alterado e por quem.
- **Apoiar revisão operacional**: Essa página é importante para rastrear comportamento e conferir responsabilidades.
- **Ler origem da mudança**: Método, status, rota, path e IP ajudam a reconstruir o evento.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o rastreio de ações.*
