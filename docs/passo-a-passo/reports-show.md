# Passo a Passo: Prévia de Relatório

## O que é possível executar:
- **Visualizar o relatório pronto**: A pessoa vê exatamente como o formulário está sendo renderizado.
- **Conferir a quantidade de páginas**: O resumo técnico informa quantas páginas existem antes da impressão.
- **Voltar para a lista de relatórios**: O botão de retorno leva de volta para a central.
- **Abrir o editor de células**: Quando disponível, o fluxo permite ajustar campos antes de voltar ao preview.
- **Imprimir a prévia**: O botão de impressão chama a saída visual do documento.
- **Inspecionar bordas e nomes**: O toggle técnico ajuda a conferir alinhamento e nome técnico dos campos.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a visualização final do relatório.*
