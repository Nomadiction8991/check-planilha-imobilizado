# Passo a Passo: Erros de Importação

## O que é possível executar:
- **Ler o título da tela**: O topo mostra “Erros de importação podem ser tratados aqui.”
- **Conferir o contexto**: Se houver igreja, administração ou importação selecionada, isso aparece abaixo do título.
- **Voltar para a origem**: O botão do topo muda conforme o modo e pode levar para nova importação ou para produtos.
- **Baixar o CSV de correção**: O botão principal gera o arquivo de correção para tratar fora da tela.
- **Ler a tabela de erros**: A tabela mostra as colunas Código, Descrição CSV, Erro e Situação.
- **Ler o código e a linha CSV**: Cada item traz o código do produto e, quando existe, a linha CSV abaixo.
- **Ler a mensagem de erro**: A coluna de erro mostra o motivo exato da falha.
- **Marcar resolvido**: Quando há permissão, o checkbox da coluna Situação troca entre Pendente e Resolvido.
- **Usar a paginação**: Se houver muitas falhas, a paginação aparece no rodapé.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a triagem de falhas.*
