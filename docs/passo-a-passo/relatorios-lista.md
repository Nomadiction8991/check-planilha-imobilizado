# Passo a Passo: Central de Relatórios

## O que é possível executar:
- **Selecionar a igreja**: A lista só aparece depois que a igreja é escolhida.
- **Carregar os relatórios**: O botão de carregar busca os formulários disponíveis.
- **Limpar a busca**: O botão de limpar devolve a tela para o estado inicial.
- **Abrir a prévia ou a posição**: Cada linha leva para a página correta do relatório.
- **Identificar o tipo de saída**: O código, a descrição e o título ajudam a evitar abrir a opção errada.
- **Ver quantos itens existem**: A coluna de itens mostra o peso ou volume do relatório.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a seleção da saída correta.*
