# Passo a Passo: Gestão Administrativa

## O que é possível executar:
- **Filtrar administrações**: Procurar por ID, descrição ou CNPJ.
- **Criar uma administração**: Abrir o formulário de cadastro novo.
- **Editar um registro**: Ajustar dados estruturais já existentes.
- **Excluir quando permitido**: Remover uma administração, respeitando confirmação e permissão.
- **Manter a base para outros módulos**: O cadastro serve como referência para igrejas, usuários e importações.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e a base administrativa do sistema.*
