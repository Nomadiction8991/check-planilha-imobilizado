# Passo a Passo: Verificação (Checklist)

## O que é possível executar:
- **Selecionar a igreja**: O primeiro filtro define o recorte do checklist.
- **Buscar itens**: O campo de busca localiza bem por código, descrição, dependência, tipo ou status.
- **Filtrar por dependência, tipo e status**: Os filtros avançados refinam a lista antes da marcação.
- **Marcar verificado**: O checkbox de verificação sinaliza que o item foi conferido.
- **Marcar etiqueta**: O checkbox de etiqueta sinaliza que o item precisa de impressão.
- **Escrever observação**: O campo de observação registra nota rápida sem sair da linha.
- **Editar cadastro**: O botão de lápis leva para a edição do produto.
- **Ir para etiquetas**: Quando a igreja está selecionada, o botão de etiquetas abre o fluxo de impressão.
- **Salvar automaticamente**: A tela envia as alterações em background conforme o usuário marca os campos.

*Nota: Esta documentação foi detalhada para cobrir os fluxos completos e o uso em conferência real.*
