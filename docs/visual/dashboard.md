# Visual: Dashboard

## Descrição Estética:
- **Layout**: Tela limpa com cards de resumo e atalhos rápidos dependendo do perfil. O dashboard funciona como sala de entrada do sistema, então ele precisa reunir informação sem virar uma parede de dados.
- **Composição**: A área principal costuma ser dividida em blocos curtos, permitindo leitura de cima para baixo sem esforço. O espaço em branco ao redor dos cards ajuda a separar os pontos de atenção.
- **Hierarquia**: Os elementos mais importantes aparecem no topo, e as ações de navegação ficam sempre próximas do que o usuário provavelmente fará em seguida. O visual tenta encurtar o caminho entre “cheguei” e “sei o que fazer”.
- **Cores**: Segue o padrão do sistema com ênfase em contraste para acessibilidade. O uso de cor costuma ser reservado para destaque de estado, cartão de atalho ou aviso pontual, sem poluição visual.
- **Responsividade**: Elementos se adaptam para visualização em tablets e desktops. Em dispositivos menores, os blocos se empilham e ainda assim preservam o entendimento da ordem das informações.
